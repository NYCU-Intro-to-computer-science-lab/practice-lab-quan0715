# p3: 小朋友走樓梯 (Climbing Stairs)

## 題目描述

小明要爬一個有 n 階的樓梯。他每次可以選擇爬 1 階或 2 階。

請問小明有多少種不同的方法可以爬到第 n 階？

## 數學背景

這其實是經典的**費波那契數列（Fibonacci Sequence）**問題：
- 爬到第 1 階：只有 1 種方法（爬 1 階）
- 爬到第 2 階：有 2 種方法（爬 1+1 階，或爬 2 階）
- 爬到第 n 階：可以從第 (n-1) 階爬 1 階，或從第 (n-2) 階爬 2 階
- 因此：`ways(n) = ways(n-1) + ways(n-2)`

## 輸入格式

一個正整數 n (1 ≤ n ≤ 40)，代表樓梯的階數

## 輸出格式

輸出一個整數，表示爬到第 n 階的方法數

## 範例

### 輸入 1
```
3
```

### 輸出 1
```
3
```

### 說明
爬到第 3 階有 3 種方法：
1. 1階 + 1階 + 1階
2. 1階 + 2階
3. 2階 + 1階

---

### 輸入 2
```
5
```

### 輸出 2
```
8
```

### 說明
- `ways(1) = 1`
- `ways(2) = 2`
- `ways(3) = ways(2) + ways(1) = 2 + 1 = 3`
- `ways(4) = ways(3) + ways(2) = 3 + 2 = 5`
- `ways(5) = ways(4) + ways(3) = 5 + 3 = 8`

---

### 輸入 3
```
1
```

### 輸出 3
```
1
```

## 限制與要求

- 時間限制：1 秒
- 記憶體限制：256 MB
- **必須使用遞迴**: 你的程式碼中必須包含名為 `climb` 的遞迴函數
- **禁止使用迴圈**: 不可使用 `for`、`while`、`do` 等迴圈關鍵字

## 提示

- Base cases:
  - `climb(1) = 1` （只有 1 種方法）
  - `climb(2) = 2` （有 2 種方法）
- Recursive case:
  - `climb(n) = climb(n-1) + climb(n-2)` （n ≥ 3）
- 函數簽章建議：`int climb(int n)`

## 注意

這是標準的 Fibonacci 數列，第 n 階的方法數等於第 (n+1) 個 Fibonacci 數。
